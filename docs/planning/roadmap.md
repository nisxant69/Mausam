# ClimaPH Roadmap

This roadmap outlines the planned features and development milestones for the ClimaPH project.

## Milestones

### Phase 1 – Setup & Core Functionality
- [x] Initialize Next.js project with TypeScript  
- [x] Set up TailwindCSS and ShadCN UI components  
- [x] Configure ESLint, Prettier, and project Git repository  
- [x] Implement fetching weather data by city (OpenWeatherMap API)  
- [x] Display basic weather information on the dashboard  

### Phase 2 – UI & UX Enhancements
- [ ] Design weather dashboard cards and components  
- [ ] Add search history feature  
- [ ] Implement temperature unit toggle (Celsius/Fahrenheit)  
- [ ] Add dark mode support  
- [ ] Display weather icons based on conditions  

### Phase 3 – Testing & Documentation
- [ ] Add unit tests for API calls and UI components  
- [ ] Improve error handling and fallback UI  
- [ ] Write detailed documentation (architecture, features, setup guides)  
- [ ] Review and optimize app performance  

### Phase 4 – Deployment & Production
- [ ] Prepare production-ready build  
- [ ] Deploy to Vercel or preferred hosting  
- [ ] Set up GitHub project board and branch protection rules  
- [ ] Maintain and monitor app performance and issues

## Notes
- Features may be adjusted based on testing feedback.  
- Documentation and GitHub Issues will be kept up-to-date throughout development.
