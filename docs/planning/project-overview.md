# ClimaPH Project Overview

**Project Name:** ClimaPH

**Purpose:**  
ClimaPH is a weather dashboard application that allows users to fetch and view real-time weather information for any city worldwide. The app aims to provide a clean, modern, and responsive interface for easy access to weather data.

**Goals:**  
- Provide accurate and up-to-date weather information using the OpenWeatherMap API.  
- Present weather data in a visually appealing and easy-to-read dashboard.  
- Enable additional functionality like search history, temperature unit conversion, and dark mode.  
- Maintain a scalable, modular, and maintainable codebase using Next.js, TypeScript, TailwindCSS, and ShadCN UI components.

**Target Users:**  
- General users looking for quick and reliable weather information.  
- Tech-savvy users who prefer a modern and responsive web application.  
- Anyone needing a lightweight and easy-to-use weather dashboard.

**Key Features:**  
- Fetch weather data by city.  
- Display weather conditions, temperature, humidity, and wind speed.  
- Search history to quickly access previously searched cities.  
- Dark mode toggle and temperature unit conversion.  
- Mobile-friendly and responsive design.  

**Tech Stack:**  
- **Frontend:** Next.js (App Router), TypeScript, TailwindCSS, ShadCN UI components  
- **API:** OpenWeatherMap API  
- **Version Control:** Git + GitHub (main/dev branches, feature branches, issues)  
- **Testing:** Unit testing with React Testing Library and Jest (optional)
