# Weather Dashboard

The Weather Dashboard is the main interface of ClimaPH where users can view real-time weather information for selected cities.

## Features

- Displays the current weather condition (e.g., sunny, rainy, cloudy).  
- Shows temperature in Celsius or Fahrenheit.  
- Displays additional data: humidity, wind speed, and feels-like temperature.  
- Shows an appropriate weather icon based on the current conditions.  
- Responsive design to fit desktop, tablet, and mobile screens.  

## Components

- **WeatherCard** – Displays weather information for a single city.  
- **DashboardGrid** – Layout component for arranging multiple WeatherCards.  

## Notes

- Future enhancements: Add weekly forecast view, animated weather icons, and city favorites.
