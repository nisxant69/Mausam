# Search History Feature

ClimaPH keeps track of previously searched cities to provide users quick access to recent searches.

## Features

- Stores last 5–10 searched cities (configurable).  
- Clickable history items to quickly fetch weather for that city again.  
- Local storage is used to persist history between sessions.  

## Components

- **HistoryList** – Displays list of recent searches.  
- **ClearHistoryButton** – Allows users to clear all stored history.  

## Notes

- Future enhancement: Sync search history with user accounts if authentication is implemented.
