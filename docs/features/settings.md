# Settings

The Settings feature allows users to customize their experience in ClimaPH.

## Features

- **Temperature Unit Toggle** – Switch between Celsius and Fahrenheit.  
- **Dark Mode Toggle** – Switch between light and dark themes.  
- **Optional Future Settings**:  
  - Default city selection  
  - Language selection (i18n)  

## Components

- **SettingsPanel** – Container for all toggle switches and settings controls.  
- **ToggleSwitch** – Reusable component for any on/off setting.  

## Notes

- Settings preferences are saved in local storage to persist between sessions.  
- Ensure toggles update the UI in real-time without requiring page reload.
